# Shopizer Admin Panel Documentation

Complete technical documentation for Shopizer admin panel (Angular).

## 📚 Documentation Files

- **[TECH-STACK.md](./TECH-STACK.md)** - Angular technology stack
- **[ARCHITECTURE.md](./ARCHITECTURE.md)** - Frontend architecture and patterns

## 🚀 Quick Start

```bash
npm install
ng serve
```

Access: http://localhost:4200

## 🔑 Key Technologies

- **Angular 11**
- **Nebular 6.2.0** UI framework
- **TypeScript 4.0.8**
- **RxJS** for state management
- **ng2-smart-table** for data grids

## 📖 Features

- Store management
- Product catalog management
- Order management
- Customer management
- Content management (CMS)
- User & permission management
